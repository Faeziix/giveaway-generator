User identified that TOF lead magnets were "not enticing enough" — need to be "sexy and broad" with the goal being purely impressions.

**Why:** The existing system treated all lead magnets the same regardless of distribution channel. It had no framework for the promotional post itself, which is the make-or-break piece for TOF virality on LinkedIn.

**How to apply:** When client goal is TOF/awareness/impressions, always recommend the TOF Viral Post playbook and generate both the resource AND the LinkedIn post copy. Reference Evan Carroll / Linear examples (7 analyzed posts, 101-761 comments) as the benchmark. Prioritize broad appeal over niche specificity at TOF.
