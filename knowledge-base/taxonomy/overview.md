# Lead Magnet Taxonomy — The 6-Category Framework

## The Categories

| # | Category | Verb | Goal | What the Prospect Gets |
|---|----------|------|------|----------------------|
| 1 | **Educate** | Learn | Build authority | Knowledge, understanding, awareness |
| 2 | **Enable** | Do | Reduce effort | A tool that does work for them |
| 3 | **Accelerate** | Start faster | Skip the blank page | A starting point they can customize |
| 4 | **Diagnose** | Understand their situation | Personalized insight | Clarity on where they stand |
| 5 | **Connect** | Access people & exclusivity | Belonging, FOMO | Community, events, early access |
| 6 | **Validate** | Justify decisions | Confidence through proof | Evidence, data, social proof |

---

## Category Details

### 1. Educate (Structured Content)
**Goal:** Teach something. Build authority and trust.

| Sub-type | Formats |
|----------|---------|
| Written | Ebooks, guides, whitepapers, reports, articles, case studies, swipe files, cheat sheets, playbooks |
| Visual | Infographics, flowcharts, diagrams, slide decks, carousels |
| Audio/Video | Webinars, mini-courses, tutorials, podcast episodes, video breakdowns |
| Curated | Resource lists, reading lists, tool stacks, directories, "best of" compilations |

### 2. Enable (Tools & Utilities)
**Goal:** Help them DO something. Reduce effort.

| Format | Examples |
|--------|----------|
| Calculators & Estimators | ROI calculators, pricing estimators, savings calculators |
| AI-powered tools | Custom GPTs, chatbots, generators (subject line, headline analyzer) |
| Browser extensions / Plugins | Workflow enhancers, scrapers, overlays |
| Spreadsheet tools | Trackers, dashboards, budget planners with built-in formulas |
| Software trials / Freemium | Limited access to your product |

### 3. Accelerate (Templates & Frameworks)
**Goal:** Give them a starting point. Skip the blank page.

| Format | Examples |
|--------|----------|
| Document templates | Email templates, proposal templates, SOPs, scripts |
| Spreadsheet templates | Budgets, project plans, content calendars, trackers |
| Design templates | Canva templates, social media templates, presentation decks |
| Notion / Airtable templates | Dashboards, wikis, CRMs, workflow systems |
| Code templates | Boilerplates, starter kits, config files |
| Frameworks / Models | Decision frameworks, mental models, strategy canvases |

### 4. Diagnose (Assessment & Diagnosis)
**Goal:** Help them understand where they stand. Personalized insight.

| Format | Examples |
|--------|----------|
| Quizzes | "What type of X are you?", knowledge tests |
| Scorecards / Audits | "Rate your website", "Grade your email strategy" |
| Self-assessments | Maturity models, readiness checklists |
| Benchmarking tools | "How do you compare to industry averages?" |
| Diagnostic surveys | Identify gaps, weaknesses, opportunities |

### 5. Connect (Community & Access)
**Goal:** Ongoing value through people and exclusivity.

| Format | Examples |
|--------|----------|
| Private communities | Slack groups, Discord servers, Facebook groups |
| Membership / gated content | Resource vaults, video libraries |
| Events access | Workshops, masterminds, office hours, AMAs |
| Early / beta access | New features, products, content before public release |
| Newsletters | Curated insights, exclusive analysis |

### 6. Validate (Social Proof & Evidence)
**Goal:** Build trust through results. Help them justify decisions.

| Format | Examples |
|--------|----------|
| Case studies | Detailed customer success stories |
| Research reports / Original data | Industry surveys, benchmarks, trend reports |
| Comparison guides | Product comparisons, vendor evaluations |
| ROI documentation | Before/after analyses, outcome summaries |

## Decision Matrix: Which Category to Use

### By Buyer Stage

| Buyer Stage | Primary Categories | Secondary |
|-------------|-------------------|-----------|
| **Awareness** (don't know you yet) | Educate, Diagnose | Connect |
| **Consideration** (evaluating solutions) | Validate, Enable | Accelerate |
| **Decision** (ready to buy/implement) | Accelerate, Enable | Validate |

### By Business Goal

| Goal | Best Categories |
|------|----------------|
| Grow email list fast | Educate, Diagnose (quizzes are viral) |
| Attract high-quality leads | Enable, Accelerate (action-takers self-select) |
| Build authority / thought leadership | Educate, Validate |
| Nurture existing leads | Connect, Educate (courses) |
| Drive product adoption | Enable (free tools), Accelerate (templates) |
| Stand out from competitors | Validate (original data), Enable (unique tools) |

### By Resource Level

| Resources Available | Best Categories |
|--------------------|----------------|
| Low budget, solo creator | Educate (checklists, cheat sheets), Accelerate (templates) |
| Medium budget, small team | Enable (calculators), Diagnose (quizzes), Validate (case studies) |
| High budget, full team | Enable (custom tools, extensions), Connect (community), Validate (research reports) |
