# Industry-to-Format Matrix

For each industry: typical pain points, best-fit lead magnet formats, example titles, and which Linear reference to adapt from.

---

## B2B SaaS

**Typical pain points:** Long sales cycles, lead quality, churn, onboarding friction, proving ROI to stakeholders
**Target audience:** Marketing managers, ops leads, founders, product managers

| Category | Best Formats | Example Titles |
|----------|-------------|----------------|
| Enable | Calculator | "Our CAC Payback Calculator & Pipeline Leak Finder" |
| Diagnose | Quiz/Assessment | "The SaaS Growth Readiness Scorecard We Use With Every New Client" |
| Accelerate | Template Collection | "12 High-Retention Onboarding Sequences From $10M+ ARR Companies" |
| Validate | Benchmark Report | "Our SaaS Benchmarks: Real Data From 200+ Companies We've Scaled" |
| Accelerate | Swipe File | "150+ Cold Email Openers That Booked 3,000+ Demos Last Year" |
| Accelerate | Checklist | "Our Bulletproof Product Launch Checklist (Used for 40+ Releases)" |

**Adaptation notes:**
- ROAS Calculator → replace ROAS with CAC:LTV, ad metrics with funnel stage conversion rates
- 470+ Hooks → "150+ Cold Email Openers" or "200+ Feature Announcement Headlines"
- Editing Review Checklist → "Product Launch Checklist" or "Feature Release QA Checklist"
- Always anchor titles to specific results: ARR numbers, demo counts, churn reduction %

## How to Use This Matrix

1. Identify the client's industry (or closest match)
2. Look at the pain points — confirm they match the client's situation
3. Pick 2-3 formats from the table based on the client's resources and buyer stage
4. Read the adaptation notes to find the Linear reference to start from
5. Read the referenced playbook for step-by-step creation guidance
6. Apply principles from `principles.md` to strengthen the concept
